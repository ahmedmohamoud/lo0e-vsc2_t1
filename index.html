  • Fork deze repository naar jouw eigen github account

  • Volg de instructies!!!!!

1. Naam aanpassen

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de master branch zit.

  • Vervang bovenaan de pagina "Naam van student" door jouw eigen naam (id=naam)
  • Commit deze wijziging

2. Afbeeldingen toevoegen

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de master branch zit.

  • Maak een nieuwe branch: afbeeldingen
  • Controleer dat je in deze branch zit
  • Zoek een passende afbeelding voor dit hoofdstuk (id=afbeeldingen), voeg deze afbeelding toe in de map images.
  • Pas de src aan van de afbeelding van dit hoofdstuk aan, zodat de nieuw toegevoegde afbeelding zichtbaar is.
  • Commit deze wijziging
  • Ga terug naar de master branch.
  • Merge de aanpassingen van afbeeldingen naar master.
  • Push de master branch naar github

3. Een conflict oplossen

Winnaar: Chuck Norris

Chuck Norris

  • Zorg dat je in de master branch zit
  • Begin een nieuwe branch: chuck
  • Pas de hierboven aangegeven winnaar (id=winnaar) aan naar: Chuck Norris
  • Pas de src van de afbeelding (id=winnaar-image) aan naar: images/chuck-norris.jpg
  • Commit deze wijziging.

Bruce Lee

  • Ga terug naar de master branch
  • Begin een nieuwe branch: bruce
  • Pas de hierboven aangegeven winnaar (id=winnaar) aan naar: Bruce Lee
  • Pas de src van de afbeelding (id=winnaar-image) aan naar: images/bruce-lee.jpg
  • Commit deze wijziging.

Conflict!

  • Ga terug naar de master branch
  • Refresh de pagina en check of Niemand de winnaar is
  • Merge de aanpassingen van bruce naar master.
  • Controleer of superman de winnaar is (refresh de pagina).
  • Merge de aanpassingen van chuck naar master.
  • Conflict!

    Je hebt nu een conflict in de header van dit article.

  • Zorg dat Chuck Norris de winnaar is.
  • Commit de merge
  • Push de master branch naar github
